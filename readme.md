# MERIDIAN TELEMETRICS — Signal Anomaly Classifier

> *Can a machine learn to hear what shouldn't be there?*

A production-grade ML pipeline that classifies deep-space radio transmissions as **ANOMALOUS** or **BACKGROUND NOISE** using a stacked ensemble model trained on 12,000 synthetic signal events.

## Model Architecture

**Stacked Ensemble** (2026 scikit-learn pattern):

```
Input (17 features)
    │
    ▼
StandardScaler
    │
    ▼
┌──────────────────────────────────────┐
│  Base Learners (trained with 3-fold  │
│  out-of-fold cross-validation)       │
│                                      │
│  RandomForest(120 trees, depth=10)   │
│  GradientBoosting(80, lr=0.1, d=4)  │
└──────────────────────────────────────┘
    │  out-of-fold probability predictions
    ▼
LogisticRegression (meta-learner)
    │
    ▼
P(anomalous) → threshold → ANOMALOUS / BACKGROUND
```

The threshold (0.9981) is chosen to maximise F1 on the validation precision-recall curve — not hardcoded at 0.5.

## Features

### Raw (13)

| Feature | Description |
|---|---|
| `frequency_mhz` | Carrier frequency in MHz |
| `bandwidth_khz` | Spectral width in kHz |
| `duration_sec` | Transmission length in seconds |
| `snr_db` | Signal-to-noise ratio (dB) |
| `drift_rate_hz_s` | Doppler drift rate — non-zero implies a moving source |
| `pulse_interval_sec` | Time between repeating pulses (0 = non-repeating) |
| `encoding_density` | Information density relative to baseline (1.0) |
| `bearing_deg` | Azimuth of arrival (0–360°) |
| `elevation_deg` | Elevation angle (0–90°) |
| `polarization_ratio` | Ratio of circular to linear polarisation |
| `spectral_kurtosis` | Non-Gaussianity of spectrum (Gaussian = 3.0) |
| `prime_interval_flag` | Whether pulse intervals follow prime spacing (bool) |
| `hydrogen_line_delta` | Absolute deviation from 1420.405 MHz |

### Engineered (4)

| Feature | Formula | Rationale |
|---|---|---|
| `snr_per_bw` | `snr_db / bandwidth_khz` | Narrow + strong = suspicious |
| `density_snr_prod` | `encoding_density × snr_db.clip(0)` | Doubly anomalous signal |
| `near_h_line` | `hydrogen_line_delta < 0.01 MHz` | Binary proximity flag |
| `has_pulse` | `pulse_interval_sec > 0` | Binary periodicity flag |

## Dataset

Generated by `src/generate_dataset.py` — fully reproducible with `seed=1974`.

```
Total rows   : 12,000
Train / Test : 9,600 / 2,400 (80/20 stratified)
Anomaly rate : 22.0%  (2,640 anomalous)
Background   : 78.0%  (9,360 background)
```

**Background signals** are modelled on natural radio sources, RFI, and satellite spillover: wide bandwidth, low SNR, random bearing, no pulse periodicity, Gaussian spectral kurtosis (~3.0).

**Anomalous signals** encode the properties that make a signal structurally suspicious to a radio astronomer: clustering on the 1420.405 MHz hydrogen line, prime-spaced pulse intervals, encoding density 11× above baseline, consistent bearing, non-Gaussian spectral kurtosis.
